# Interview Trick #1

## SQL Execution Order
### Question

```sql
SELECT name
FROM Employee
WHERE salary > AVG(salary);
```

## Is this valid

Answer: Valid.

Wrong.

Because

WHERE executes before AVG.

Correct

```sql
SELECT *
FROM Employee
WHERE salary >
(
SELECT AVG(salary)
FROM Employee
);
```

# Order of Execution

FROM

↓

WHERE

↓

GROUP BY

↓

HAVING

↓

SELECT

↓

ORDER BY

↓

LIMIT

This alone answers many interviews.
